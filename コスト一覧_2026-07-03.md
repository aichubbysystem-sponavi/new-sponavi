========================================================
## A. 従量課金（使うほど増える）★ここが危険
========================================================

### A-1. Google Cloud ─ Places API (New) searchText  ← 17万円の正体
課金対象。コード内で使う場所は【4箇所】。

| # | 使用箇所(API) | 機能 | 単価/回 | 1操作の最大費用 |
|---|---|---|---|---|
| 1 | grid-ranking | 順位計測(グリッド) | ¥4.8 | ¥1,176（7×7圏外/1KW）★主犯 |
| 2 | keyword-volume | KWボリューム調査 | ¥4.8 | ¥48（10KW/1実行） |
| 3 | sync-coordinates | 座標取得 | ¥5.3 | ¥5〜6/店（取れない店のみ・一度きり） |
| 4 | competitors | 競合分析 | ¥6.0 | ¥6/実行 |

▼ grid-ranking を「操作単位」で見た最大費用（＝実際の爆弾）
| 操作 | 最大費用 |
|---|---|
| 1店舗フル計測（8KW・実測+圏外多） | 約¥1,700 |
| 全店舗一括計測（505店） | ¥30〜50万/回 ⚠️絶対注意 |
| 未計測のみ一括（400店） | ¥25〜40万/回 ⚠️絶対注意 |

※単価が変わる理由 = FieldMask（取得フィールド）の違い
  displayNameのみ=Pro(¥4.8) / +住所座標=Enterprise(¥5.3) / +評価=Atmosphere(¥6.0)

### A-2. Google Cloud ─ Maps JavaScript API（地図表示）
- 使用箇所【2箇所】: grid-ranking画面, レポート画面([shopId])
- 地図を1回表示 = 1ロード ≒ ¥1
- 現状は請求書上¥0（無料枠内）。ただし表示回数が増えれば課金し得る → 監視対象

### A-3. Anthropic Claude API（api.anthropic.com）★Google Cloud外・別請求
- AI生成系すべて。使用箇所【15箇所】:
  monthly-analysis / chat / auto-reply / analyze / generate-description /
  generate-post(3) / bulk-generate / reply-suggest / survey /
  pmax/summary-text / report各種
- 単価: モデルとトークン量による従量
- ※今回のGoogle Cloud 17万には含まれない。Anthropicから別途請求が来る
